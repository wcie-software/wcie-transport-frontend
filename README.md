# WCIE Transport Frontend
WCIE Transport allows church members to request rides to church

## Features
- Verifies the user's phone number.
- Allows the user to find their address (limited to Canada.)
- Saves the user's details to Firestore database.
- Directs to Fillout form to get request details.
